Como jogar. Concorra a prêmios grandiosos com a Quina: basta marcar de 5 a 15 números dentre os 80 disponíveis no volante e ficar na torcida.

Aqui estão os 5 números aleatórios.

from random import randint num = list() while True: sort = randint(0, 80) if sort not in num: num.append(sort) if len(num) == 5: break print(sorted(num))
