A Lotomania é fácil de jogar e de ganhar: basta escolher 50 números e então concorrer a prêmios para acertos de 20, 19, 18, 17, 16, 15 ou nenhum número.

Aqui estão os 50 números aleatórios.

from random import randint num = list() while True: sort = randint(0, 99) if sort not in num: num.append(sort) if len(num) == 50: break print(sorted(num))
